## 为什么使用框架

框架的好处：

1. 组件化
   高度组件化可以使工程易于维护
2. 生态环境
   现在主流的前端框架都自带生态，不管是数据流管理架构还是 UI 库都有成熟的解决方案
3. 开发效率
   解放了开发者手动操作 dom
4. 天然分层
   mcv 和 mvvm 模式都能帮助我们进行分层，代码解藕更易于读写
