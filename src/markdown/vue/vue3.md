# 哈哈

## vue-cli

安装： yarn global add @vue/cli
升级： yarn global upgrade --latest @vue/cli
创建项目： vue create hello-world-project (只有 babel + eslint)
可视化: vue ui

## 添加插件

vue ui 的 GUI 操作面板中添加：

vue-router

vuex

vue-cli-plugin-axios
